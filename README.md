# meap

A fullstack meal prep and nutrition tracking app built to solve a real problem: staying healthy while juggling school, work, and the gym.

**Live App:** [meapprep.vercel.app](https://meapprep.vercel.app/)

---

## Why I Built This

As a master's student trying to build muscle, I struggled to track what I was eating and what ingredients I had in stock. Meal prepping was my solution for staying consistent, but managing recipes, nutrition, and inventory was chaotic. So I built meap to make it effortless.

This project represents 3 months of learning fullstack development while balancing school and work. It's my first production app built from scratch.
